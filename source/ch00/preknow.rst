- 基本数据

    - 数字,字串,布尔 基本和其它脚本语言类同
    - `NaN`, `null` ~ 空值

- 全局方法

    - `eval()` ~ 将给入的字串,尝试执行
    - `isFinite()`,`isNaN()` 是否有限和空
    - `parseInt()`,`parseFloat()`,`toString()` 进行类型转换
    - `escape()`, `unescape()` 对 url 进行编码和相反,,


- 基本对象

 - Number  数字
 - String   字串
 - Array    数组
 - Math     数学
 - Date     日期

各自包含一堆内置的各种常用处理操作.

然后,一般教程,会进行复杂的文档对象(Dom) 的说明,,,
- 这是特指 html 文档对象的操作
